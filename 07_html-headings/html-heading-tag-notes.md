<!-- +++++++++++++++ HTML Heading Tag +++++++++++++++ -->
=> The Heading tag is used to display the headings in HTML.
=> A Total of 6 headings are there:-
<h1> Main Heading </h1> (Most Important)
<h2> Sub Heading </h2>
<h3> Section Heading </h3>
<h4> Subsection </h4>
<h5> Minor Heading </h5>
<h6> Smallest Heading </h6> (Least Important)

=> <h1> is the most important heading.
=> <h6> is the least important heading.
=> Mostly we use h1, h2, and h3.
