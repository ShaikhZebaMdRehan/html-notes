<!-- ++++++++++ Get Familiar With Chrome Developer Tool ++++++++++ -->
1. What is Chrome Developer Tools?
=> A Chrome tool in Google Chrome to inspect and debug websites.
=> Helps you view HTML, CSS, JavaScript, and network performance.

2. How to Open DevTools?
=> You can open DevTools by: Click on the three dots (top-right corner of Chrome) -> Select More tools -> Click on Developer Tools
=> Right-click → Inspect
=> Keyboard shortcuts: (Windows) Ctrl + Shift + I and (Mac) Cmd + Option + I

3. Main Panels You Use
=> Elements → View & edit HTML/CSS Code.
=> Styles → Modify CSS live.
=> Console → Run JS Code & view errors.
=> Network → Check loaded files & speed.

4. Why Use DevTools?
=> Debug layouts.
=> Change styles.
=> Test responsive design.
=> Find errors in scripts.
=> Measure performance.

5. Quick Practical Uses
=> Inspect an element → see what CSS affects it.
=> Turn off a CSS property to test effects.
=> Resize window to test mobile layout.

6. Useful Tips
=> You can copy CSS directly from DevTools and paste it into an original file.
=> You can edit the page without uploading new files.
=> DevTools shows computed styles — what the browser actually uses.

7. View Page Source or Edit Code
=> View HTML code: Inspect Element or View Page Source (Ctrl + U).
=> Edit via Inspect Element (temporary changes only).
