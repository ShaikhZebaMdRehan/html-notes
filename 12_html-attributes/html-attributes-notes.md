<!-- +++++++++++++++ HTML Attributes +++++++++++++++ -->
=> All HTML elements can have attributes.
=> Attributes add extra information to HTML elements.
=> They are written inside the start tag.
=> Usually in name/value pairs, e.g., name="value".

<!-- +++++++++++++++ Common HTML Attributes +++++++++++++++ -->
=> The href=" " attribute of the <a> tag is used to specify the URL.
=> The target=" " attribute of the <a> tag specifies where to open the link.
=> The title=" " attribute gives extra information about an element (shown as a tooltip on hover).
=> The src=" " attribute of the <img> tag defines the path of the image.
=> The width=" " and height=" " attributes of <img> set the size of the image.
=> The alt=" " attribute of <img> provides alternative text if the image cannot load.
=> The style=" " attribute is used to add inline CSS styles to an element, such as color, font, size, and more.
=> The lang=" " attribute of the <html> tag declares the language of the web page.
