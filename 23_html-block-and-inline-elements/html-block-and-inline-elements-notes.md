<!-- +++++++++++++++ Div and Span +++++++++++++++ -->
1. Div
=> <div> is used as a container for other HTML elements.
=> It is a block-level element and starts on a new line.
=> A block element takes full width by default.
=> Multiple <div> elements can be used on one page.
=> Changing display to inline-block makes <div> elements appear side by side.
=> <div> is widely used for creating web pages—almost 95% of layout structures are created using <div>.
=> Using a class with <div> makes styling and control easier with CSS.

2. Span
=> <span> is an inline-level container used for other HTML elements.
=> Inline elements do not start on a new line.
=> They take width according to their content.
