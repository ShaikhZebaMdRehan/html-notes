<!-- +++++++++++++++ HTML Text Formatting Part 01 +++++++++++++++ -->
=> HTML provides several tags to format and highlight text on a webpage.
1. Bold & Strong
=> <b> → Makes text bold (visual only)
=> <strong> → Shows important text (bold + semantic meaning)

2. Italic, Emphasis, Address & Cite
=> <i> → Italic text (visual only)
=> <em> → Emphasized text (italic + meaning)
=> <address> → Shows contact/author information (italic by default)
=> <cite> → Used for titles of books, artwork, movies, etc

3. Underline & Inserted
=> <u> → Underlined text
=> <ins> → Inserted text (usually underlined)

4. Abbreviation
=> <abbr> → Shows a tooltip on hover and adds a dotted underline to the text

5. Mark
=> <mark> → Highlights text (yellow by default), but we CSS instead of <mark>

6. Subscript & Superscript
=> It is used in mathematical formulas or chemical expressions.
=> <sub> → Subscript text (e.g., H₂O)
=> <sup> → Superscript text (e.g., x²)

7. Deleted / Striked
=> <del> → strikethrough on text
=> <strike> → Old tag, not supported in HTML5

8. Small & Big
=> <small> → Smaller text
=> <big> → Bigger text (not supported in HTML5)

<!-- +++++++++++++++ HTML Text Formatting Part 02 +++++++++++++++ -->
1. <blockquote> – Long Quotations → indent the text

2. <q> – Short Quotations → add quotation marks

3. <pre> – Preformatted Text → displays text exactly as typed including spaces and line breaks

<!-- +++++++++++++++ Marquee Tag +++++++++++++++ -->
Note: <marquee> is not supported in HTML5. It is old and deprecated, but still works in some browsers.
=> But, mostly we use the direction and scrollamount attribute.
=> Marquee Tag Attributes:
1. direction → scroll direction
2. scrollamount → speed of scrolling
3. bgcolor → background color
4. height & width → size of marquee
5. behavior → scrolling style
6. scrolldelay → delay in milliseconds
7. loop → number of repeats
8. hspace & vspace → adds horizontal and vertical spacing
