<!-- +++++++++++++++ Download the VS Code +++++++++++++++ -->
1. Go to Google Chrome, search for VS Code, and download it as per your OS Windows, MAC, or Linux.
2. I am downloading for Windows 64-bit.
3. Open the VS Code folder and set up VS Code.
4. Click on I accept the agreement and click on Next.
5. Click on Next.
6. Click on Next.
7. Tick on check box: Create a desktop icon, Add "Open with code" action to Windows Explorer file context menu, Add "Open with code" action to Windows Explorer directory context menu, Add to PATH (requires shell restart), click on Next.
8. Click on Next.
9. Click on Finish.

<!-- +++++++++++++++ VS Code Brief Tour +++++++++++++++ -->
1. For color theme: VS Code File -> preferences -> color theme.
2. Change the font size of the VS Code: VS Code file -> preferences -> settings -> change the font size.
3. Open the folder through CMD command: click on the file URL -> write CMD -> write the command (code), -> enter.

<!-- +++++++++++++++ Useful extensions in VS Code +++++++++++++++ --> 
1. Live Server - Ritwick Dey: Go to extension in VS Code -> Download Live server
2. Bracket Pair Colorizer - Coenraads: Go to extension in VS Code -> Download Bracket Pair Colorizer
=> It is used to give color to the brackets.
=> This extension is not useful in HTML and CSS.
=> But it is useful in JavaScript Code.

3. Prettier - Code Formatter - Esben Petersen: Go to extension in VS Code -> Download Prettier - Code Formatter
=> It is used to convert your code to a prettier format.
