<!-- +++++++++++++++ Basic of HTML +++++++++++++++ -->
<!DOCTYPE html>   -> this tells the browser you are using HTML5
<html lang="en">  -> root of the HTML document
<head>   -> container for the metadata
    <title>My First Page</title>  -> page title
</head>
<body>   -> contains all the code in the body tag, such as h1, p, img, etc
    <h1>Hello Worl</h1>   -> heading tag
    <p>hello world</p>   -> paragraph tag
</body>
</html>

<!-- +++++++++++++++ Quick Points +++++++++++++++ -->
=> HTML documents start with <!DOCTYPE html>.
=> <!DOCTYPE html> is not case-sensitive.
=> <!DOCTYPE html> appears only once at the top of the HTML page.
=> The document starts with <html> and ends with </html>.
=> Visible content is inside <body></body>.
