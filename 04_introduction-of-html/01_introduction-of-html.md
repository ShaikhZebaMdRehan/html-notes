<!-- +++++++++++++++ Introduction of HTML +++++++++++++++ -->
Q1. What is the full form of HTML?
=> HTML stands for Hyper Text Markup Language.

Q2. Who invented HTML?
=> Tim Berners-Lee wrote the first version of HTML in 1993, and now we have been using HTML5 since 2014.

Q3. What is the use of HTML?
=> HTML (Hyper Text Markup Language ) is the standard markup language used to create and design webpages.
=> It describes the structure and content of web documents using elements and tags.
=> Markup language: A markup language is a computer language that uses tags to define elements.
=> It is predefined tags.
=> An HTML element is defined by a start tag, some content, and an end tag:
<tagname> Content goes here... </tagname>

Q4. Why do we use it?
=> We use HTML to build and structure webpages, ensuring content like text, images, and links are properly formatted and displayed on web browsers.

Q5. With which name do we save an HTML file?
=> An HTML file is saved with .html at the end of the file name.

Q6. First HTML File
=> index.html
=> It is the default name for a website's homepage.

Q7. HTML Tag
=> An HTML tag is a container for content or other HTML tags.
<p> This is a Paragraph </p>
This is a Paragraph => Content
<p></p> => Element or Paragraph Tag
