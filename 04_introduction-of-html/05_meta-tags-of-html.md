<!-- +++++++++++++++ HTML Meta Tags Information +++++++++++++++ -->
1. <meta charset="UTF-8">
<!-- Character encoding -->
=> It defines the character encoding of the page.
=> UTF-8 stands for UCS Transformation Format-8.
=> UTF-8 is the most common encoding used on the web.
=> UTF-8 ensures correct text display on all browsers.
=> HTML5 recommends using UTF-8 encoding.
=> It supports all Unicode characters and is compatible with ASCII.
=> ASCII stands for American Standard Code for Information Interchange.
=> UTF-8 encoding means every character can be readable on every webpage.
=> To display an HTML page correctly, a web browser must know which character set to use.

2. <meta name="viewport" content="width=device-width, initial-scale=1.0">
<!-- Viewport for Responsive Design -->
=> The viewport is the visible area of a webpage on a device.
=> It changes based on device size (mobile, tablet, desktop).
=> Older websites were designed only for desktops with fixed widths.
=> HTML5 introduced the <meta name="viewport"> tag for responsive design.
=> It tells the browser how to control page width and zoom.
=> content="width=device-width sets the page width to the device screen width.
=> initial-scale=1.0 sets the default zoom level when the page loads.

3. <meta http-equiv="X-UA-compatible" content="IE=edge">
<!-- Browser Compatibility -->
=> It is used for browser compatibility.
=> It ensures the webpage displays correctly in Internet Explorer.

4. <meta name="author" content="Your Name">
<!-- Specifies the Author of the webpage. -->
=> It defines the author name of the webpage
=> It tells browsers and search engines who created the website.

5. <meta name="description" content="Page descripton for search engine">
<!-- Brief Description of the webpage -->
=> It provides a brief description of the webpage.
=> The description is written inside the content attribute.
=> It is used for SEO and search engine results.

6. <meta name="keywords" content="html, web development, tutorial">
<!-- Keywords related to the website -->
=> It defines keywords related to the website for Ranking on Google.
=> Keywords are written inside the content attribute.
=> It was used for SEO and search engine ranking.
