Q1. What is a website? Types of websites.
=> A place connected to the internet, where a company, organization, etc, puts information that can be found on the World Wide Web.
=> There are 2 types of websites:-
1. Static website: In a static website, there is no response.
Eg: Wikipedia

2. Dynamic website: In a dynamic website, there is a response.
Eg: Amazon, Flipkart, etc

Q2. What is a Domain?
=> A Domain name is a unique address where all other users can navigate the website with the help of the internet.
Eg: .com, .in, .shop, .org, etc
=> Types of Domain Name:
 .com -> Commercial
 .org -> Organization
 .net -> Netwrok
 .edu -> Education
 .gov -> Government
 .mil -> Military
 .co -> Comapny
 .info -> Information
 .biz -> Business
 .in -> India

Q3. What is Hosting? Types of Hosting.
=> An internet hosting service that allows users and companies to make their website accessible on the internet.
Eg: GoDaddy, Hostinger, HostGator, etc
=> There are 2 types of Hosting:-
1. Shared: In Shared Hosting, we can host multiple websites.
2. Dedicated: In Dedicated Hosting, we can host only one website.

Q4. What is a Responsive Website?
=> A website that is accessible on both, i.e, Laptop, Desktop, and Mobile, is called a Responsive website.

Full Form:
URL: Uniform Resource Locator
HTTP: Hyper Text Transfer Protocol
WWW: World Wide Web

=> WWW is about communication between web clients and web servers.
=> Web clients (such as browsers) send HTTP requests to web servers, and web servers respond with HTTP responses.
