<!-- +++++++++++++++ HTML Tables +++++++++++++++ -->
=> HTML tables allow web developers to arrange data into rows and columns.
<th> → table header
<tr> → table row
<td> → table data
<thead> → table head
<tbody> → table body

Key Features & Attributes:
1. Colspan & Rowspan: Merge columns or rows.
2. Caption: Add Table heading using <caption>.
3. Column Styling: <colgroup> and <col> to style specific columns; use span to specify number of columns.
4. Borders: border property; border-collapse: collapse to avoid double borders; border-radius for rounded corners; border-color to set color.
5. Table Size: To set the width of a table via style on <table>.
6. Padding & Spacing: padding for cell padding, border-spacing for space between cells.
7. Zebra Stripes: Alternate row/column coloring using tr:nth-child(even).
