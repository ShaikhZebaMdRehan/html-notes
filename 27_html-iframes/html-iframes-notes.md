<!-- +++++++++++++++ HTML Iframes +++++++++++++++ -->
=> An <iframe> is used to display another webpage inside a webpage.
=> It defines an inline frame and embeds an external document within the current page.
=> width and height attributes set the iframe size.
=> Always add a title attribute to the <iframe> for accessibility, as screen readers use it to describe the iframe’s content.
=> By default, <iframe> have a border (which can be removed).
=> An <iframe> can be used as a target for links using the name attribute.
Example: Showing YouTube, Google Map, or another site inside your page.

<!-- ++++++++++ How to Get an Iframe Link (Step-by-Step) ++++++++++ -->
1. From YouTube
Step 1: Open any YouTube video
Step 2: Click Share
Step 3: Click Embed
Step 4: Copy the iframe code (it starts with <iframe ...>)
    
2. From Google Maps
Step 1: Open Google Maps
Step 2: Search for any place
Step 3: Click Share
Step 4: Select Embed a map
Step 5: Copy the iframe code (it starts with <iframe ...>)
