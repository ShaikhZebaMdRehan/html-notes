<!-- ++++++++++ HTML Links - Create Single page Bookmark ++++++++++ -->
=> HTML bookmarks help users jump to specific sections of a webpage.
=> They are useful for long or single-page websites.
=> Bookmarks are created using the id attribute.
=> When the link is clicked, the page scrolls automatically to that section.
Example:
Link on same page: <a href="#C4">Jump to Chapter 4</a>
Create bookmark: <h2 id="C4">Chapter 4</h2>

<!-- ++++++++++ HTML Links - Create Multipage Bookmark ++++++++++ -->
=> HTML multi-page bookmarks link to a specific section on another webpage.
=> They are useful for websites with long content on different pages.
=> Bookmarks are created using the id attribute.
=> When the link is clicked, it directly goes to that page.
Example:
Link on another page: <a href="home.html">Jump to Chapter 4</a>
Create bookmark: <h2 id="home">Chapter 4</h2>s
