<!-- ++++++++++ HTML Links OR Anchor Tags ++++++++++ -->
=> To add links in HTML, we use the <a> tag, which is called the anchor tag.
=> The href=" " attribute of the <a> tag is used to specify the URL.
=> The target=" " attribute of the <a> tag specifies where to open the link.
=> The title=" " attribute gives extra information about an element (shown as a tooltip on hover).
<a href="https://google.com" target="_blank" title="Go to Google">Google</a>

<!-- ++++++++++ HTML Links - The target Attribute ++++++++++ -->
=> By default, links open in the same tab (target="_self").
=> The target attribute decides where the linked page will open.
=> But mostly, we use the target attribute with target="_blank".
=> The target attribute values:
1. _self - same tab (default)
2. _blank - new tab
3. _parent - parent frame
4. _top - full browser window

<!-- ++++++++++ HTML <a> Download Attribute ++++++++++ -->
=> The HTML download attribute makes the linked file download when the user clicks the link on the hyperlink.
