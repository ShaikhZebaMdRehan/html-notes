<!-- +++++++++++++++ HTML Images +++++++++++++++ -->
=> <img> tag is used to add images to your web pages.
=> The <img> tag is a self-closing tag.
=> The Two main attributes:
src → image path
alt → alternate text shown if the image cannot load; useful for accessibility and screen readers
width & height → set image size
Example: <img src=" " alt=" " width=" " height=" ">

<!-- +++++++++++++++ Absolute and Relative URL +++++++++++++++ -->
1. Absolute URL → links to an external image
Example: <img src="https://www.w3schools.com/images/img_girl.jpg">
Note: May have copyright issues; image can change or be removed.

2. Relative URL - links to an image within your website.
Example: <img src="img_girl.jpg"> or <img src="/images/img_girl.jpg">
Tip: It is almost always best to use relative URLs. They will not break if you change the domain.
